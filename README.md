# MMM-Scheduler


### Install & Configuration ###
___
    cd ~/MagicMirror/modules
    git clone https://github.com/jun01270/MMM-Scheduler.git
    cd MMM-Scheduler
    npm install

### Using the module ###
Add it to the modules array in the `config/config.js` file:

```javascript
{
module: "MMM-Scheduler",
position: "top_right",
config: {
	}
},
```
